# Cafe Billing System

A simple console-based cafe billing system developed in Python.  
This program allows users to select items from a menu, specify quantities, and generate a final bill.

---

## Features
- Display menu with prices
- Add multiple items to cart
- Specify quantity for each item
- Generate and print the final bill
- Exit option
